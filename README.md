## 🌐 Despliegue con GitHub Pages

Para desplegar tu portafolio:

1. Ve a **Settings** de tu repositorio en GitHub
2. En la sección **Pages**, selecciona:
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: /docs
3. Guarda los cambios
4. Tu sitio estará disponible en: `https://[tu-usuario].github.io/profile`
